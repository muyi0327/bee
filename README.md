# Bee
基于vue2.0开发的一套，手机端组件库

## 准备工作
    

## 安装
    npm install bee -S

## 快速上手
    import Vue from 'vue';
    import Bee from 'bee';

    Vue.use(Bee);

    // or
    <script src="node_modules/bee-mobile/dist/bee.js"></sciprt>
    <script src="node_modules/bee-mobile/dist/bee.css"></sciprt>

    // or
    import Icon from 'bee-mobile/packages/icon';
    Vue.use(Icon);

    <bee-icon fill="orange"></bee-icon>

## 流览器兼容

    Android 4.2+
