# bee-icon

bee-icon is a mobile icon component for bee library

## Installation

First, install bee-icon from wanda npm

    npm install bee-icon or npm install bee

Then use it in vue;

    import Icon from 'bee/packges/icon';

    var vm = new Vue({
        el: '#app',
        template: '<div><bee-icon></bee-icon></div>',
        components:{
            'w-icon': Icon
        }
    });
