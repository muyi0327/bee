# wui-icon

wui-icon is a mobile icon component for wui library

## Installation

First, install wui-icon from wanda npm

    npm install wui-icon or npm install wui

Then use it in vue;

    import Icon from 'wui/packges/icon';

    var vm = new Vue({
        el: '#app',
        template: '<div><w-icon></w-icon></div>',
        components:{
            'w-icon': Icon
        }
    });
